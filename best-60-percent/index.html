‚ö†Ô∏è Before you buy a 60%: Make sure you're comfortable with:
  • No dedicated arrow keys (use Fn + WASD or IJKL)
  • No function row (F1-F12 on number keys)
  • Learning curve of 1-2 weeks for muscle memory

If you're unsure, consider a 65% or 75% instead. Same compactness, keeps arrows separate.
