How to Build a Custom Keyboard

Your first custom build: what you need, how to do it, and what to avoid. No soldering required (if you don't want to).

1

Test Your PCB

Before installing anything, test that the PCB works. This saves you from disassembling later if there's an issue.

  1. Plug the PCB into your computer (USB-C or USB-A)
  2. Go to a keyboard tester site (search "keyboard tester")
  3. Use tweezers or a paperclip to touch the two metal holes where a switch would go
  4. The tester should register a keypress
  5. Test multiple spots, especially edge keys
üí° Pro Tip: If a key doesn't register, check if the hotswap socket is damaged. You can usually fix it with a quick solder touch-up, but email the vendor if it's a new kit.
2

Install Stabilizers

Stabilizers go under the big keys: Spacebar, Enter, Shift, Backspace, and sometimes the + key on the numpad.

  1. Unclip the stabilizer wire from the housing
  2. Slide the housing into the PCB mount points
  3. Clip the wire back in
  4. Repeat for all stabilizer positions
‚ö†Ô∏è Common Mistake: Make sure the stabilizer wire is facing the correct direction. The curve should face toward the spacebar, not away from it.
3

Install Switches

Time to populate the board. With hot-swap, you just push them in. (If soldering, this is where you'd solder each pin).

  1. Align the switch pins with the holes in the PCB
  2. Make sure both pins are straight (if bent, gently straighten with pliers)
  3. Push straight down with firm pressure until you hear/feel a click
  4. The switch should sit flush with the plate
  5. Do all switches, leaving the stabilized keys for last
üí° Pro Tip:>When pushing switches, support the back of the PCB with your other hand. This prevents bending the board.
4

Test Everything Again

Before closing up the case, test every single key again. It's much easier to fix issues now than after assembly.

  • Plug in the keyboard
  • Open a keyboard tester
  • Press every key, one by one
  • Test the spacebar, enter, shift (stabilized keys)
  • Check that no keys are "chattering" (registering twice)
5

Put It in the Case

Now assemble the case. Process varies by case type, but generally:

  1. Place any case foam (if your kit came with it)
  2. Lower the completed PCB/plate assembly into the case
  3. Secure with screws (usually comes with the case)
  4. Don't overtighten - just snug
6

Install Keycaps

The satisfying final step. Be gentle - keycaps can crack if forced.

  1. Start with stabilized keys first (they're hardest)
  2. Align the + shape with the stem
  3. Push straight down with even pressure
  4. Do the rest in whatever order works for you
üí° Pro Tip: If a keycap feels stuck, don't force it. The stabilizer might not be seated right. Remove the cap and reseat the stabilizer wire.
