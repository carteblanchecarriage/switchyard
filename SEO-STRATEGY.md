## On-Page SEO Checklist

### Title Tags
Format: `Primary Keyword - Secondary Keyword | Keebshelf`
- Home: "Mechanical Keyboard Group Buys & In-Stock Drops | Keebshelf"
- Product: "Keychron Q1 - Custom Mechanical Keyboard [In Stock] | Keebshelf"
- Category: "Mechanical Keyboard Cases - 60%, 65%, 75% Layouts | Keebshelf"
- Blog: "Custom Mechanical Keyboard Build Guide [2025] | Keebshelf"

### Meta Descriptions
- 150-160 characters
- Include primary keyword
- Call to action
- Unique per page

### URLs
- `/mechanical-keyboard-guide` (not `/blog/post-123`)
- `/keyboards/keychron-q1` (not `/product?id=456`)

### Headers (H1-H6)
- One H1 per page
- Include keyword in H1
- Use H2-H6 for structure

### Internal Linking
- Link related products
- Link to pillar content
- Breadcrumb navigation
- Footer links to main categories

### Image Optimization
- Descriptive filenames: `keychron-q1-mechanical-keyboard.jpg`
- Alt text: "Keychron Q1 custom mechanical keyboard with aluminum case"
- WebP format
- Lazy loading
- Structured data
